+Веб-приложение для расчета графика смен, часов и дохода, а также для ведения личного журнала смен.
+
+## Возможности
+
+- Расчет по графикам `2/2`, `5/2`, `1/3` и пользовательскому циклу.
+- Учет часов за смену и стоимости часа.
+- Учет дополнительных смен по дате и часам.
+- Расчет дохода за месяц и заработка к текущей дате (с учетом часового пояса).
+- Отдельная страница `Личный журнал смен`:
+  - обычные и дополнительные смены по дням;
+  - настройка длительности обычной и доп. смены;
+  - статистика часов за месяц и за все время.
+- PWA-режим (установка как приложение, офлайн-кэш).
+
+## Страницы
+
+- `index.html` — основной калькулятор.
+- `journal.html` — личный журнал смен.
+
+## Локальный запуск
+
+1. Откройте папку проекта в терминале.
+2. Запустите локальный сервер:
+
+```powershell
+python -m http.server 8080
+```
+
+3. Откройте в браузере:
+
+```text
+http://localhost:8080
+```
+
+Важно: PWA (service worker, установка приложения) работает через `http://localhost` или `https`, не через `file://`.
+
+## Структура проекта
+
+- `index.html`, `styles.css`, `script.js` — калькулятор смен.
+- `journal.html`, `journal.css`, `journal.js` — личный журнал смен.
+- `manifest.webmanifest`, `sw.js` — PWA.
+- `icons/` — иконки приложения.
+
+## Примечание по данным
+
+Сейчас данные пользователя хранятся локально в браузере (`localStorage`/`sessionStorage`).
+Для входа с разных устройств нужен backend + облачная БД.
